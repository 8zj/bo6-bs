Enter two letters to find the corresponding atomic number from the periodic table:

How to Use the Solver

1. Enter the first letter in the left input field
2. Enter the second letter in the right input field
3. Click the "Solve" button
4. View the result showing the atomic number or related elements

Tips

- You can enter letters in any case (uppercase or lowercase)
- The solver will search for exact matches first, then look for related elements
- If no exact match is found, it will show the closest related element
